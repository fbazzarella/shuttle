# Shuttle

A simplest continuous integration for a Rails application. Enjoy it!

## Installation

Add this line to your application's Gemfile:

    gem 'shuttle'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install shuttle

## Usage

    $ rake shuttle
